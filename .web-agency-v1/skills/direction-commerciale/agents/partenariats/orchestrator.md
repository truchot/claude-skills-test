# Orchestrateur Partenariats

Coordination des partenariats stratégiques et de l'écosystème.

## Agents Disponibles

| Agent | Responsabilité |
|-------|----------------|
| `strategie-partenariats` | Vision et plan de partenariats |
| `evaluation-partenaires` | Évaluation et scoring |
| `modeles-collaboration` | Frameworks de collaboration |
| `suivi-partenariats` | Performance et suivi |

| Mots-clés | Agent |
|-----------|-------|
| stratégie partenariat, alliance | `strategie-partenariats` |
