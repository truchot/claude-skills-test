# Agent Objectifs CA

Définition des objectifs de chiffre d'affaires.

## Responsabilité

Fixer et suivre les objectifs de revenus.

## Inputs

- Historique CA
- Pipeline commercial
- Capacité de production
- Objectifs de croissance

## Outputs

- Objectifs annuels/trimestriels
- Répartition par segment
- Milestones de suivi
- Scénarios (base, optimiste, pessimiste)

## Structure Objectifs

| Période | Objectif | Base | Stretch |

| T1 | 300k€ | 250k€ | 350k€ |
| T2 | 350k€ | 300k€ | 400k€ |

| T4 | 400k€ | 350k€ | 450k€ |
| **Année** | **1.35M€** | **1.15M€** | **1.55M€** |

## Métriques de Suivi

- Forecast accuracy : Écart réel vs prévu
- Pipeline coverage : Pipeline / Objectif (cible > 3x)
- Win rate : Affaires gagnées / Propositions
