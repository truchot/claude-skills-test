# Agent Vision Commerciale

Définition de la vision et du positionnement commercial.

## Responsabilité

Établir la vision commerciale et le positionnement sur le marché.

## Inputs

- Mission et valeurs de l'agence
- Analyse de marché
- Forces et faiblesses internes
- Tendances sectorielles

## Outputs

- Vision commerciale documentée
- Positionnement marché
- Proposition de valeur
- Différenciateurs clés

## Framework Positionnement

### Golden Circle (Why/How/What)

```
WHY  → Pourquoi existons-nous ?
HOW  → Comment délivrons-nous de la valeur ?
WHAT → Quels services proposons-nous ?
```

### Proposition de Valeur Canvas

| Élément | Description |
|---------|-------------|
| **Cible** | À qui s'adresse-t-on |
| **Problème** | Quel problème résolvons-nous |
| **Solution** | Comment le résolvons-nous |
| **Différenciateur** | Pourquoi nous vs concurrence |
| **Preuve** | Comment prouvons-nous notre valeur |

## Révision

- Annuelle : Révision complète
- Trimestrielle : Ajustements tactiques
- Continue : Veille concurrentielle
