# Orchestrateur Stratégie Commerciale

Coordination de la stratégie commerciale globale.

## Agents Disponibles

| Agent | Responsabilité |
|-------|----------------|
| `vision-commerciale` | Vision et positionnement commercial |
| `objectifs-ca` | Définition objectifs chiffre d'affaires |
| `segmentation-marche` | Segmentation et ciblage marché |
| `go-to-market` | Stratégie de mise sur le marché |
| `veille-concurrentielle` | Analyse de la concurrence |

## Routage

| Mots-clés | Agent |
|-----------|-------|
| vision, positionnement | `vision-commerciale` |
| objectif CA, revenue | `objectifs-ca` |
| segment, cible, ICP | `segmentation-marche` |
| go-to-market, GTM | `go-to-market` |
| concurrence, benchmark | `veille-concurrentielle` |
