# Agent Optimisation Coûts

Stratégie d'optimisation et réduction des coûts.

## Responsabilité

Identifier et piloter les leviers d'optimisation des coûts.

## Inputs

- Structure de coûts actuelle
- Benchmark secteur
- Objectifs de marge
- Contraintes opérationnelles

## Outputs

- Plan d'optimisation
- Quick wins identifiés
- Économies estimées
- Risques et mitigation

## Catégories de Coûts

| Catégorie | % Budget | Potentiel Optim |
|-----------|----------|-----------------|
| Salaires | 60% | Faible (qualité) |
| Sous-traitance | 15% | Moyen |
| Outils/Licences | 8% | Moyen |
| Locaux | 7% | Moyen (remote) |
| Marketing | 5% | Variable |
| Autres | 5% | Moyen |

## Leviers d'Optimisation

### Court Terme (Quick Wins)

- [ ] Audit licences inutilisées
- [ ] Renégociation fournisseurs
- [ ] Optimisation cloud (right-sizing)
- [ ] Réduction frais généraux

### Moyen Terme

- [ ] Automatisation tâches répétitives
- [ ] Mutualisation ressources
- [ ] Amélioration productivité (outils, process)
- [ ] Mix CDI/Freelance optimisé

### Long Terme

- [ ] Offshoring/Nearshoring partiel
- [ ] Développement outils internes
- [ ] Économies d'échelle (croissance)

## Règles

- Ne jamais compromettre la qualité client
- Impliquer les équipes dans la démarche
- Mesurer avant/après chaque action
- Réinvestir une partie des économies
