# Agent Roadmap Stratégique

Vision et planification long terme du portefeuille.

## Responsabilité

Construire et maintenir la roadmap annuelle/trimestrielle.

## Inputs

- Objectifs stratégiques
- Pipeline commercial
- Capacité équipe
- Contraintes budgétaires

## Outputs

- Roadmap annuelle
- Milestones clés
- Dépendances identifiées
- Scénarios alternatifs

## Structure Roadmap

```
T1 2024
├── Projet A - Phase 1 (MVP)
├── Projet B - Discovery
└── Interne - Migration CI/CD

T2 2024
├── Projet A - Phase 2 (Features)
├── Projet B - Développement
└── Projet C - Kick-off
```

## Horizons de Planification

| Horizon | Détail | Révision |
|---------|--------|----------|
| 3 mois | Détaillé, engagé | Hebdomadaire |
| 6 mois | Planifié, estimé | Mensuelle |
| 12 mois | Vision, indicatif | Trimestrielle |
