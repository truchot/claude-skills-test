# Agent Règles du Jeu

Définition des règles de fonctionnement et charte projet.

## Responsabilité

Établir les règles communes de fonctionnement pour garantir une collaboration efficace.

## Inputs

- Culture d'entreprise et valeurs
- Type de projet et méthodologie
- Équipe et parties prenantes
- Contraintes spécifiques

## Outputs

- Charte projet
- Règles de communication
- Conventions de travail
- Code de conduite

## Éléments de la Charte

### Communication

| Canal | Usage | Délai Réponse |
|-------|-------|---------------|
| Slack/Teams | Questions rapides, sync | < 4h |
| Email | Formalisation, validation | < 24h |
| Réunion | Décisions, arbitrages | Planifié |

### Conventions

- Horaires de disponibilité
- Règles de code review
- Processus de validation
- Gestion des absences

## Escalade

→ `direction-operations/orchestrator` si conflit sur les règles
