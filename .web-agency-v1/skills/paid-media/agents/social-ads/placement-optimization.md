## Optimisation Manuelle (Si Nécessaire)

### Quand Exclure des Placements

| Situation | Action |
|-----------|--------|
| CPM très élevé sur placement | Exclure si CPA non rentable |
| Objectif Brand premium | Exclure Audience Network |
| B2B LinkedIn | Feed uniquement |
| Format unique | Placements compatibles |

### Breakdown Analysis

```
Analyser dans Ads Manager:
Breakdown → Delivery → Placement

Métriques à comparer:
• CPM par placement
• CTR par placement
• CPA par placement
• ROAS par placement
```
