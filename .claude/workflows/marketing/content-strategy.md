# Workflow: Stratégie de Contenu

## Objectif
Définir une stratégie de contenu alignée avec les objectifs business.

## Prérequis
- Objectifs business définis
- Accès aux données analytics
- Budget et ressources identifiés

## Étapes

### 1. Audit Contenu Existant
**Responsable**: Content manager
**Durée**: 1-2 semaines

- [ ] Lister tous les contenus existants
- [ ] Analyser les performances (trafic, conversions)
- [ ] Identifier les gaps
- [ ] Catégoriser: garder, rafraîchir, supprimer

**Output**: Audit contenu avec recommandations

### 2. Définition des Personas
**Responsable**: Marketing + Sales

- [ ] Identifier les segments cibles
- [ ] Créer les fiches personas
- [ ] Définir les pain points
- [ ] Mapper le parcours client
- [ ] Valider avec l'équipe commerciale

**Output**: 2-4 personas documentés

### 3. Recherche de Mots-Clés
**Responsable**: SEO / Content manager
**Skills**: `seo/contenu`

- [ ] Brainstormer les thèmes principaux
- [ ] Utiliser les outils SEO (Ahrefs, SEMrush)
- [ ] Analyser les intentions de recherche
- [ ] Prioriser par volume × faisabilité
- [ ] Créer les clusters thématiques

**Output**: Liste de mots-clés priorisés

### 4. Architecture Pillar-Cluster
**Responsable**: Content manager

- [ ] Définir les pages pillar (thèmes principaux)
- [ ] Identifier les clusters (sous-thèmes)
- [ ] Planifier le maillage interne
- [ ] Créer la carte de contenu

**Output**: Architecture contenu validée

### 5. Calendrier Éditorial
**Responsable**: Content manager

- [ ] Définir la fréquence de publication
- [ ] Répartir les contenus par semaine/mois
- [ ] Assigner les auteurs
- [ ] Définir les deadlines
- [ ] Configurer l'outil de suivi

**Output**: Calendrier éditorial 3-6 mois

### 6. Définition des KPIs
**Responsable**: Marketing
**Skills**: `analytics-marketing`

- [ ] Définir les objectifs par type de contenu
- [ ] Configurer le tracking
- [ ] Créer le dashboard de suivi
- [ ] Planifier les revues mensuelles

**Output**: Dashboard KPIs

## KPIs Recommandés

| Métrique | Cible |
|----------|-------|
| Trafic organique | +X%/mois |
| Temps sur page | > 3 min |
| Conversions | X/mois |
| Backlinks acquis | X/contenu |

## Points d'Escalade

| Situation | Action |
|-----------|--------|
| Pas de ressources rédaction | Externaliser ou réduire fréquence |
| Performances faibles après 3 mois | Audit et pivot stratégique |
| Changement objectifs business | Réviser la stratégie |
