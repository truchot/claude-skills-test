## Copywriting - Formules

### AIDA (Attention, Interest, Desire, Action)
```
ATTENTION: Accroche qui capte
INTEREST: Problème et enjeux
DESIRE: Solution et bénéfices
ACTION: CTA clair
```

### PAS (Problem, Agitation, Solution)
```
PROBLEM: Identifier le problème
AGITATION: Amplifier la douleur
SOLUTION: Présenter la solution
```

### 4U (Urgent, Unique, Ultra-specific, Useful)
- **Urgent** - Créer l'urgence
- **Unique** - Différencier
- **Ultra-specific** - Être précis
- **Useful** - Apporter de la valeur

## Blog Article Structure

```markdown
# Titre optimisé SEO (H1)

**Introduction** (hook + promesse)

## Sous-titre 1 (H2)
Contenu avec valeur actionnable...

### Point détaillé (H3)
Exemples, données, citations...

## Sous-titre 2 (H2)
...

## Conclusion
Récap + CTA
```

### Checklist Article Blog
- [ ] Titre avec mot-clé (< 60 caractères)
- [ ] Meta description (150-160 caractères)
- [ ] Introduction avec hook
- [ ] Structure H2/H3 logique
- [ ] Images optimisées avec alt
- [ ] Liens internes (2-3 minimum)
- [ ] CTA en conclusion
- [ ] Longueur adaptée (1500-2500 mots)

## Social Media

### Formats par Plateforme

| Plateforme | Format optimal | Ton |

| Twitter/X | Thread, one-liner | Concis, punchy |

| TikTok | Vidéo courte | Authentique, fun |

### Hook Social Media
```
❌ "Voici comment améliorer votre marketing"
✅ "J'ai augmenté mes conversions de 312% en changeant UNE chose..."
```

## Email Marketing

### Structure Email
```
OBJET: < 50 caractères, curiosité/bénéfice
PREVIEW: Complète l'objet

SALUTATION personnalisée

ACCROCHE (1-2 phrases)

CORPS (valeur, histoire)

CTA (un seul, clair)

SIGNATURE
```

### Types d'Emails
- **Welcome series** - Onboarding
- **Newsletter** - Valeur régulière  
- **Promotional** - Offres, lancements
- **Nurturing** - Éducation, confiance
- **Re-engagement** - Réactivation

## Métriques Clés
