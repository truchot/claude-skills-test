## Frameworks de Copywriting

### AIDA (Classique)
```
A - Attention  → Hook accrocheur
I - Intérêt    → Problème identifié
D - Désir      → Solution et bénéfices
A - Action     → CTA clair
```

### PAS (Problem-Agitate-Solution)
```
P - Problem   → Identifier le problème
A - Agitate   → Amplifier la douleur
S - Solution  → Présenter la solution
```

### BAB (Before-After-Bridge)
```
B - Before → Situation actuelle douloureuse
A - After  → Vision idéale
B - Bridge → Comment y arriver (produit)
```

### 4U (Headlines)
```
Urgent   → Crée l'urgence
Unique   → Différenciant
Ultra-specific → Précis et concret
Useful   → Bénéfice clair
```

## Formules Éprouvées

### Titres qui Fonctionnent

| Type | Formule | Exemple |
|------|---------|---------|
| **How-to** | Comment [obtenir résultat] | Comment doubler vos leads en 30 jours |
| **Listicle** | [X] façons de [bénéfice] | 7 façons de réduire vos coûts marketing |
| **Question** | Vous voulez [désir] ? | Vous voulez plus de clients ? |
| **Négatif** | Arrêtez de [erreur] | Arrêtez de gaspiller votre budget pub |
| **Chiffre** | [X]% de [résultat] | 47% de conversions en plus |
| **Secret** | Le secret de [résultat] | Le secret des marques qui cartonnent |

| Catégorie | Mots |
|-----------|------|

| **Résultat** | Garanti, Prouvé, Testé, Efficace |
| **Gratuité** | Gratuit, Offert, Bonus, Cadeau |
| **Nouveauté** | Nouveau, Révolutionnaire, Inédit |
| **Sécurité** | Protégé, Sécurisé, Confiance |

## Règles d'Or du Copy

1. **Bénéfice > Feature** : Vendez le résultat, pas le produit
2. **Vous > Nous** : Parlez au lecteur, pas de vous
3. **Simple > Complexe** : Un enfant de 10 ans doit comprendre
4. **Spécifique > Vague** : "47%" plutôt que "beaucoup"
5. **Actif > Passif** : "Économisez 100€" plutôt que "100€ sont économisés"
6. **Court > Long** : Chaque mot doit mériter sa place
