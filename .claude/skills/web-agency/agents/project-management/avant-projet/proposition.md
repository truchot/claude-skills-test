## Bonnes Pratiques

### Ce qui fait une bonne proposition

| Faire | Ne pas faire |
|-------|--------------|
| Reformuler le besoin client | Copier-coller le brief |
| Mettre en avant les bénéfices | Lister les fonctionnalités |
| Être concis et percutant | Faire 50 pages |
| Personnaliser pour le client | Template générique |
| Proposer des options | Offre unique figée |

### Longueur recommandée

| Type de projet | Longueur |
|----------------|----------|
| Petit projet (< 10K€) | 5-10 pages |
| Projet moyen (10-50K€) | 10-20 pages |
| Gros projet (> 50K€) | 20-30 pages |

### Éléments visuels

- Inclure le logo du client
- Schémas et diagrammes pour clarifier
- Screenshots de références similaires
- Timeline visuelle pour le planning
