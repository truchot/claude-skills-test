# Conventions SEO
- Balise title unique par page (< 60 caractères)
- Meta description unique (< 160 caractères)
- Structure Hn hiérarchique : un seul H1, H2-H6 ordonnés
- Images : attribut alt descriptif, formats modernes (WebP, AVIF)
- URLs propres : slugs lisibles, pas de paramètres inutiles
- Données structurées Schema.org en JSON-LD
- Core Web Vitals : LCP < 2.5s, FID < 100ms, CLS < 0.1
