# Conventions Design Tokens
- Single source of truth : tokens définis une seule fois
- Naming : category-property-variant (ex: color-primary-500)
- CSS Custom Properties comme format de sortie
- Échelles cohérentes : spacing (4px base), typography (modular scale)
- Tokens sémantiques au-dessus des tokens primitifs
- Atomic Design : foundations → atoms → molecules → organisms → templates
