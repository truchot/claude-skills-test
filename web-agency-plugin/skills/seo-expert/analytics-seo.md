## Google Search Console Analysis

### Query Analysis
```
Top queries by clicks -> Content optimization priorities
Queries with high impressions, low CTR -> Title/meta improvement
Queries on page 2 (positions 11-20) -> Quick win targets
New queries appearing -> Content expansion opportunities
```

### Page Performance
```
Pages with declining clicks -> Content refresh needed
Pages with high impressions, low clicks -> CTR optimization
New indexed pages -> Monitor initial performance
Pages with coverage issues -> Fix crawl/index errors
```
