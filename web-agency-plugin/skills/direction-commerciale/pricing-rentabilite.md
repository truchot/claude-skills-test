# Pricing & Rentabilite - Condensed

## Modeles de Pricing

| Modele | Quand | Avantage | Risque |
|--------|-------|----------|--------|
| Forfait | Scope bien defini, MVP, refonte | Previsibilite | Scope creep |
| Regie | Scope evolutif, agile, long terme | Marge garantie | Confiance requise |
| Mixte | Discovery forfait + Build regie plafonnee + Run forfait | Equilibre | Complexite contrat |

### Arbre de decision
```
Scope bien defini ? → Oui → Client accepte prix fixe ? → Oui → FORFAIT
                                                        → Non → REGIE/MIXTE
                    → Non → REGIE ou MIXTE
```

## Chiffrage Projet

### Methodes
1. **Bottom-up** : Prix = somme(Jours x TJM) x (1 + Marge)
2. **Analogique** : Prix = Projet similaire x Facteur ajustement
3. **Parametrique** : Prix = (Nb ecrans x cout) + (Nb integrations x cout) + ...

### Marges cibles
| Type | Marge cible | Min acceptable |
|------|-------------|----------------|
| Forfait | 40% | 30% |
| Regie | 35% | 25% |
| Maintenance | 50% | 40% |
| Conseil | 50% | 35% |
| Formation | 60% | 45% |

### Buffer risque
| Risque | Buffer |
|--------|--------|
| Faible (specs claires, techno connue) | +10% |
| Moyen (specs partielles, equipe mixte) | +20% |
| Eleve (innovation, client nouveau) | +30% |

## Negociation

### Principes
- Jamais negocier le prix seul → toujours une contrepartie
- Connaitre son BATNA et celui du client

### Grille contreparties
| Remise | Contrepartie |
|--------|-------------|
| 5-10% | Paiement 30j, engagement ferme |
| 10-15% | Reduction scope 15-20% |
| 15-20% | Engagement pluriannuel |
| > 20% | Validation direction requise |

### Red lines
- Marge jamais < 25%
- Pas de travail gratuit (spec, POC)
- Acompte minimum 30%

## Analyse Rentabilite

### Metriques par projet
| Metrique | Cible | Alerte |
|----------|-------|--------|
| Marge brute | > 40% | < 30% |
| Marge nette | > 25% | < 15% |
| Ecart budget | < 10% | > 20% |
| Productivite | > 600/j | < 450/j |
| Taux facturation | > 85% | < 70% |

### Classification clients (BCG)
| Type | Marge | CA | Action |
|------|-------|-----|--------|
| Star | > 35% | > 50k/an | Developper, premium |
| Cash Cow | > 30% | Stable | Maintenir, automatiser |
| Question | < 25% | Potentiel | Renegocier |
| Dog | < 20% | < 20k/an | Renegocier ou desengager |

## Seuils : OK > 30%, Attention 25-30%, Alerte 20-25%, Critique < 20%
