# Relation Client Strategique - Condensed

## Comptes Cles

### Criteres compte cle
| Critere | Seuil |
|---------|-------|
| CA annuel | > 50k |
| Potentiel | > 100k |
| Recurrence | > 2 ans relation |
| Strategique | Reference, innovation |

### Account Plan - Structure
```
1. Situation : CA N-1, services actuels, interlocuteurs, satisfaction
2. Potentiel : Besoins identifies, budget, projets a venir, objectif CA N
3. Strategie : Positionnement vise, differenciateurs, risques
4. Actions : 3-5 actions datees avec responsable
5. KPIs : Croissance CA, NPS, part de portefeuille
```

## Satisfaction Client (NPS)

### Benchmark
| Score NPS | Interpretation |
|-----------|----------------|
| > 50 | Excellent |
| 30-50 | Bon |
| 0-30 | A ameliorer |
| < 0 | Critique |

### Moments de mesure
- Fin de projet (transactionnel)
- Annuel (relationnel)
- Post-support (transactionnel)

### Actions par segment
| Segment | Action |
|---------|--------|
| Promoteurs (9-10) | Demander reference, temoignage |
| Passifs (7-8) | Identifier leviers d'amelioration |
| Detracteurs (0-6) | Appel immediat, plan de redressement |

## Prevention Churn

### Signaux d'alerte
| Signal | Risque | Action |
|--------|--------|--------|
| NPS < 6 | Eleve | Contact immediat |
| Usage en baisse | Moyen | Analyse et contact |
| Retard paiement | Modere | Suivi commercial |
| Changement interlocuteur | Modere | RDV presentiel |
| Demande de benchmark | Eleve | Intervention direction |
| Silence > 3 mois | Moyen | Prise de nouvelles |

### Plan reactif
1. J+0 : Contact pour comprendre
2. J+2 : Proposition de solution
3. J+7 : Offre de retention si necessaire
4. J+14 : Escalade direction

### Offres de retention
Prix eleve → remise/scope | Qualite → plan amelioration | Besoin → extension | Relation → changement interlocuteur
