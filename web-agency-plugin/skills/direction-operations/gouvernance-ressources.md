# Gouvernance & Ressources - Condensed

## Gouvernance Projet

### Comitologie par taille
| Taille | Comites |
|--------|---------|
| Small (< 50k) | Weekly sync, Monthly review |
| Medium (50-200k) | Weekly sync, Bi-weekly steering, Monthly review |
| Large (> 200k) | Daily standup, Weekly steering, Bi-weekly exec, Monthly board |

### RACI - Principes
- **R** Responsable (fait), **A** Approbateur (valide), **C** Consulte, **I** Informe
- Un seul A par decision
- R et A peuvent etre la meme personne sur petits projets

### Regles de Communication
| Canal | Usage | Delai reponse |
|-------|-------|---------------|
| Slack/Teams | Questions rapides, sync | < 4h |
| Email | Formalisation, validation | < 24h |
| Reunion | Decisions, arbitrages | Planifie |

## Vision Projet - Framework

```
Pour [CLIENT/UTILISATEUR CIBLE]
Qui [BESOIN/PROBLEME]
Le [NOM DU PROJET]
Est [TYPE DE SOLUTION]
Qui [BENEFICE PRINCIPAL]
Contrairement a [ALTERNATIVE]
Notre solution [DIFFERENCIATEUR CLE]
```

### OKR Projet
```
Objective : [Qualitatif, inspirant, aligne strategie]
  KR1 : [Metrique + cible + deadline]
  KR2 : [Metrique + cible + deadline]
  KR3 : [Metrique + cible + deadline]
```

## Ressources Humaines

### Capacite Equipe
| Metrique | Cible | Alerte |
|----------|-------|--------|
| Taux utilisation | 75-85% | < 60% ou > 90% |
| Heures facturables | 70% | < 60% |

```
Capacite brute = (Effectif x Jours ouvres) - Conges - Formation
Capacite nette = Capacite brute x 0.85
```

### Staffing - Arbre de decision
```
Besoin identifie
├─ Duree < 3 mois → Freelance ou ESN
├─ Competence strategique → CDI prioritaire
├─ Budget contraint → Freelance ou montee en competence interne
└─ Besoin recurrent → CDI
```

## Coordination Inter-equipes

### Rituels
| Rituel | Frequence | Participants | Objectif |
|--------|-----------|--------------|----------|
| Stand-up cross-team | Quotidien | Tech leads | Blocages |
| Sync design-dev | Bi-hebdo | Design + Dev | Alignement |
| Review transverse | Hebdo | Tous leads | Dependances |
| Retro globale | Mensuel | Toute equipe | Amelioration |

### Dependances inter-projets
| Type | Risque | Mitigation |
|------|--------|------------|
| Ressource | Disponibilite | Allocation strategique |
