# Branding Guide - Condensed

## Plateforme de Marque

| Composant | Contenu | Livrable |
|-----------|---------|----------|
| Identite | Personnalite, valeurs, ADN visuel | Plateforme de marque |
| Langage visuel | Couleurs, typo, icono, grille | Systeme visuel |
| Ton de voix | Personnalite verbale, do's/don'ts | Guide editorial |
| Charte | Documentation, regles d'usage, templates | Brand guidelines |

## Workflow Branding

```
1. Brand Audit (existant) → Forces/faiblesses, coherence
2. Brand Identity         → Personnalite, ADN, valeurs
3. Visual Language        → Palette, typo, icono, grille
4. Tone of Voice          → Principes editoriaux, exemples
5. Brand Guidelines       → Charte complete, templates
```

## Audit de Marque - Checklist

- [ ] Analyse identite actuelle (logo, couleurs, typo)
- [ ] Evaluation coherence cross-canal (web, print, social)
- [ ] Identification forces et faiblesses
- [ ] Benchmark concurrentiel visuel
- [ ] Recommandations d'evolution

## Langage Visuel - Composants

| Element | A definir | Format |
|---------|-----------|--------|
| Palette couleurs | Primaires, secondaires, neutres, semantiques | HEX/RGB/HSL |
| Typographie | Titres, corps, UI, code | Font stack + fallbacks |
| Iconographie | Style, taille, grille, couleurs | SVG library |
| Grille | Colonnes, gouttiere, breakpoints, espacements | Tokens |

## Ton de Voix - Framework

### Axes de personnalite (curseurs 1-5)
- Formel ←→ Decontracte
- Serieux ←→ Humoristique
- Technique ←→ Accessible
- Reserve ←→ Enthousiaste

### Principes editoriaux
1. Voix active, phrases courtes
2. Jargon technique adapte a l'audience
3. Exemples concrets > abstractions
4. Coherence entre tous les canaux

### Do's and Don'ts par contexte

| Contexte | Do | Don't |
|----------|-----|-------|
| Site web | Clair, direct, benefice client | Jargon interne, superlatifs |
| Email client | Personnel, professionnel | Familier, impersonnel |
| Social media | Engage, authentique | Corporate, distant |
| Documentation | Precis, structure | Vague, desordonne |

## Charte de Marque - Structure

```
1. Mission et vision
2. Valeurs de marque
3. Logo (usage, zones, declinaisons)
4. Couleurs (palette, ratios, accessibilite)
5. Typographie (hierarchie, usage web/print)
6. Iconographie et illustration
7. Photographie (style, traitement)
8. Mise en page (grilles, marges)
9. Ton de voix
10. Exemples d'application
```

## Validation Creative

- [ ] Alignement avec la vision de marque
- [ ] Respect de l'identite (couleurs, typo, ton)
- [ ] Coherence visuelle globale
- [ ] Qualite des livrables (resolution, formats)
- [ ] Respect des standards accessibilite (contraste WCAG)
