# Strategie Marketing - Condensed

## Triptyque Fondamental (Prerequis)

```
1. Discovery   → problem-definition.md + offer-definition.md
2. Personas    → persona.md
3. Positioning → brand-positioning.md
```
**Ordre strict** : Pas de strategie sans triptyque valide.

## Positionnement & Personas

### Value Proposition Canvas
| Element | Contenu |
|---------|---------|
| Benefices cles | Ce que le client gagne |
| Messages par segment | Adaptes a chaque persona |
| Proof points | Cas clients, chiffres, certifications |
| USPs | Avantages competitifs soutenables |

### Persona - Structure
```
Nom, role, entreprise type
Objectifs / Frustrations / Motivations
Canaux preferes / Criteres de decision
Citation representative
```

### Differenciation - Axes
- Expertise technique (stack, certifs)
- Methodologie (process, outils)
- Resultats mesurables (ROI, delais)
- Experience client (accompagnement)

## Objectifs Marketing (SMART)

### Structure OKR
```
Objective : [Qualitatif, inspirant]
  KR1 : [Metrique + cible + deadline]
  KR2 : [Metrique + cible + deadline]
  KR3 : [Metrique + cible + deadline]
```

### KPIs strategiques
| Categorie | KPI | Exemple cible |
|-----------|-----|---------------|
| Notoriete | Trafic organique, mentions | +30%/trimestre |
| Acquisition | Leads qualifies, CAC | X leads, CAC < Y |
| Conversion | Taux MQL→SQL, win rate | > 20% |
| Retention | NPS, churn rate | NPS > 40 |

## Roadmap Marketing

### Structure trimestrielle
```
T1 : Fondations (positionnement, contenu pilier)
T2 : Acceleration (campagnes, partenariats)
T3 : Optimisation (A/B tests, scaling)
T4 : Consolidation (bilan, planning N+1)
```

### Budget Strategy
| Canal | % Budget indicatif |
|-------|--------------------|
| Content/SEO | 30-40% |
| Paid (Ads) | 20-30% |
| Events/PR | 10-15% |
| Outils/Tech | 10-15% |
| Design/Production | 10% |

## Brief Marketing
Contexte, Objectif (SMART), Cible (personas), Message cle, Canaux, Budget, Timeline, KPIs
