# Performance & Acquisition - Référence condensée

## Email Marketing

### Types et benchmarks
| Type          | Objectif         | Fréquence     | Open rate |
|---------------|------------------|---------------|-----------|
| Newsletter    | Engagement       | 1x/semaine    | > 25%     |
| Promotional   | Vente directe    | 2-4x/mois     | > 20%     |
| Transactionnel| Confirmation     | Sur événement | > 60%     |
| Nurturing     | Lead → client    | Séquence auto | > 30%     |

### Checklist email
- [ ] Subject < 50 car., personnalisé | Preheader 40-130 car.
- [ ] 1 CTA principal visible, design responsive mobile-first
- [ ] Ratio texte/image > 60/40, lien désinscription clair
- [ ] SPF + DKIM + DMARC configurés
- [ ] A/B test systématique (objet, CTA)

## Growth - Framework AARRR
| Étape       | Question                  | Tactiques clés                    |
|-------------|---------------------------|-----------------------------------|
| Acquisition | Comment ils arrivent ?     | SEO, social ads, partnerships     |
| Activation  | 1ère expérience positive ? | Onboarding email, quick win       |
| Rétention   | Ils reviennent ?           | Drip emails, feature updates      |
| Revenue     | Ils paient ?              | Upsell in-app, pricing tiers      |
| Referral    | Ils recommandent ?         | Parrainage, avis incentivés       |

## CRO (Conversion Rate Optimization)

### Funnel et benchmarks
```
Visiteur → Lead : 2-5% (B2B)  |  Lead → MQL : 30-40%
MQL → SQL : 40-60%            |  SQL → Client : 20-30%
```

### Éléments à tester par impact
| Élément           | Impact    | Test type    |
|-------------------|-----------|--------------|
| Value proposition | Très fort | Message test |
| CTA texte+design  | Fort      | A/B test     |
| Form (nb champs)  | Fort      | Réduction    |
| Pricing display   | Très fort | Layout test  |
| Page speed        | Fort      | Technique    |

### Outils CRO
| Outil            | Usage              | Coût     |
|------------------|--------------------|----------|
| Hotjar           | Heatmaps, recordings| Freemium|
| Microsoft Clarity| Heatmaps gratuit   | Gratuit  |
| AB Tasty         | A/B testing, FR    | Payant   |

## Diagnostic funnel rapide
| Symptôme              | Diagnostic probable     | Action              |
|-----------------------|-------------------------|---------------------|
| Trafic OK, leads bas  | LP ou offre faible      | Revoir proposition  |
| Leads OK, MQL bas     | Mauvais ciblage         | Affiner audiences   |
| MQL OK, SQL bas       | Nurturing insuffisant   | Optimiser séquences |
| SQL OK, close rate bas| Process vente/pricing   | Aligner sales/mktg  |

## Métriques clés
| Métrique  | Formule                          | Cible       |
|-----------|----------------------------------|-------------|
| CVR       | Conversions / Visiteurs          | > 2%        |
| CAC       | Coût total mktg / Nouveaux clients| < LTV/3    |
| LTV       | ARPU x Durée moyenne client      | > 3x CAC    |
| RPV       | Revenue / Visiteurs uniques      | En hausse MoM|
