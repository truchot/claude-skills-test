# Retention & Negociation

## Negociation

### Framework LAER pour objections

1. **Listen** - Ecouter l'objection sans interrompre
2. **Acknowledge** - Valider la preoccupation ("Je comprends")
3. **Explore** - Comprendre le fond ("Qu'est-ce qui vous inquiete ?")
4. **Respond** - Repondre avec valeur et preuves

### Reponses types aux objections

**"C'est trop cher"**
- Reframer : cout vs investissement, ROI projete
- Comparer : cout de ne rien faire vs cout du projet
- Ajuster : phaser le projet, MVP d'abord

**"On hesite avec un concurrent"**
- Differencier : expertise, references, qualite
- Prouver : etudes de cas similaires, temoignages
- Rassurer : garanties, SLA, accompagnement

**"Pas le bon moment"**
- Quantifier le cout d'attente
- Proposer un demarrage progressif
- Reserver un creneau pour rappel a date convenue

### Proposition commerciale - Structure

```markdown
## Proposition [Nom Client]

### 1. Comprehension du besoin (1 page)
### 2. Solution proposee (2-3 pages)
### 3. Planning previsionnel (1 page)
### 4. Investissement (1 page)
   - Option A : [scope complet]
   - Option B : [scope reduit / phase 1]
### 5. Equipe dediee (0.5 page)
### 6. References similaires (0.5 page)
### 7. Conditions et prochaines etapes
```

### Tactiques de closing

| Situation | Technique |
|-----------|-----------|
| Decision lente | Offre limitee dans le temps |
| Hesitation budget | Facilites de paiement |
| Besoin validation | Proposer un call avec decideur |
| Peur du risque | Garantie satisfaction / pilote |

## Fidelisation et retention

### Indicateurs de risque churn

| Signal | Poids | Action immediate |
|--------|-------|-----------------|
| Usage en baisse | High | Alerte + outreach |
| Tickets negatifs | High | Escalade support |
| Non-connexion 30j | High | Appel compte |
| NPS detracteur (<6) | Medium | Appel manager |
| Retard paiement | Medium | Revue commerciale |
| Non-participation events | Low | Nurturing cible |

### Programme de fidelisation

| Action | Frequence | Objectif |
|--------|-----------|----------|
| Business review | Trimestriel | Aligner objectifs |
| NPS survey | Semestriel | Mesurer satisfaction |
| Newsletter VIP | Mensuel | Maintenir engagement |
| Event/webinar invite | Trimestriel | Renforcer relation |
| Upsell review | Semestriel | Identifier opportunites |

### Detection upsell

| Signal | Opportunite |
|--------|------------|
| Trafic en hausse | Scaling infra, CDN |
| Nouveau produit | Sous-site, landing page |
| Recrutement | Formation, documentation |
| Refonte identite | Design system, charte |
| Expansion internationale | Multi-langue, i18n |

### Gestion des renouvellements

- J-90 : notification interne, revue du compte
- J-60 : contact client, bilan de satisfaction
- J-30 : envoi proposition de renouvellement
- J-15 : relance si pas de retour
- J-0 : renouvellement ou cloture

### Metriques retention

| Metrique | Cible |
|----------|-------|
| Taux renouvellement | > 85% |
| Net Revenue Retention | > 110% |
| NPS | > 40 |
| Churn mensuel | < 3% |
| Upsell rate | > 20% |
