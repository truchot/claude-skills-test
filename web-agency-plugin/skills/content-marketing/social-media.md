# Social Media - Référence condensée

## Stratégie par plateforme

| Plateforme  | Audience cible    | Format star       | Fréquence   | Objectif principal |
|-------------|-------------------|-------------------|-------------|-------------------|
| LinkedIn    | B2B, décideurs    | Carrousel, article| 3-5x/sem    | Thought leadership|
| Instagram   | B2C, 18-40 ans    | Reels, Stories    | 5-7x/sem    | Branding, engage  |
| TikTok      | Gen Z, < 30 ans   | Short video       | 1-2x/jour   | Awareness, viral  |
| X (Twitter) | Tech, news, veille| Thread, actu      | 3-5x/jour   | Veille, influence |
| Facebook    | 35+, local        | Vidéo, events     | 3-5x/sem    | Communauté, local |
| YouTube     | Tous              | Tuto, long format | 1-2x/sem    | SEO vidéo, autorité|

## Calendrier éditorial social

### Template hebdomadaire
```
Lun → Contenu éducatif / tips
Mar → Témoignage client / case study
Mer → Contenu interactif (sondage, quiz)
Jeu → Behind the scenes / culture
Ven → Contenu promotionnel / offre
Sam → UGC / communauté
Dim → Inspiration / storytelling
```

### Horaires optimaux (Europe/Paris)
| Plateforme | B2B              | B2C              |
|------------|------------------|------------------|
| LinkedIn   | Mar-Jeu 8h-10h   | -                |
| Instagram  | Mar-Ven 12h-14h  | Sam-Dim 10h-12h  |
| TikTok     | Lun-Ven 19h-21h  | Sam-Dim 14h-17h  |
| Facebook   | Mer-Ven 13h-16h  | Sam-Dim 9h-12h   |

## Community Management

### Matrice de réponse

| Type              | Délai max | Ton                  | Action              |
|-------------------|-----------|----------------------|---------------------|
| Question simple   | 1h        | Amical, utile        | Répondre            |
| Plainte publique  | 30 min    | Empathique, pro      | Répondre + DM       |
| Crise / bad buzz  | 15 min    | Factuel, transparent | Escalade immédiate  |
| Compliment        | 2h        | Chaleureux           | Remercier + engager |
| Troll             | -         | Ignorer              | Masquer si besoin   |
| Avis négatif      | 1h        | Constructif          | Répondre + suivi    |

### Templates de réponse
```
Plainte : "Merci de nous avoir alertés, [Prénom].
Nous comprenons votre frustration concernant [sujet].
Nous vous envoyons un DM pour résoudre cela rapidement."

Compliment : "Merci beaucoup [Prénom] ! Ça nous fait
très plaisir. N'hésitez pas à partager votre expérience !"
```

## Engagement Strategy

### Formule engagement rate
```
ER = (likes + commentaires + partages + saves) / reach × 100
```

### Benchmarks engagement

| Plateforme | Bon ER    | Excellent ER |
|------------|-----------|--------------|
| Instagram  | > 3%      | > 6%         |
| LinkedIn   | > 2%      | > 5%         |
| TikTok     | > 5%      | > 10%        |
| Facebook   | > 1%      | > 3%         |

### Leviers d'engagement
- [ ] Questions ouvertes en fin de post
- [ ] Sondages et quiz interactifs
- [ ] Répondre à TOUS les commentaires < 2h
- [ ] UGC : republier le contenu communauté
- [ ] Séries récurrentes (attente + fidélité)
- [ ] Collaborations avec créateurs/influenceurs

## Social Listening

### Outils recommandés
| Outil         | Usage                    | Prix       |
|---------------|--------------------------|------------|
| Mention       | Monitoring marque        | Payant     |
| Brandwatch    | Analytics avancé         | Enterprise |
| Hootsuite     | Gestion + écoute         | Payant     |
| Google Alerts | Monitoring gratuit basique| Gratuit   |
| BuzzSumo      | Contenu performant       | Payant     |

### Veille : quoi surveiller
- Mentions marque (+ variantes, typos)
- Mentions concurrents
- Mots-clés secteur / tendances
- Sentiment global (positif/négatif/neutre)
- Influenceurs qui parlent du secteur
