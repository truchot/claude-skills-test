# Content Strategy - Référence condensée

## Arborescence & Architecture de contenu

### Structure type site
```
/               → Home (hero + value prop)
├── /services   → Pages services (conversion)
├── /blog       → Hub éditorial (SEO + nurturing)
├── /cas-clients → Preuves sociales
├── /a-propos   → Confiance, équipe
├── /contact    → Conversion directe
└── /ressources → Lead magnets (guides, templates)
```

### Checklist arborescence
- [ ] Max 3 niveaux de profondeur
- [ ] Parcours utilisateur mappé par persona
- [ ] Mots-clés SEO assignés par page
- [ ] Maillage interne planifié
- [ ] Navigation testée (card sorting)

## Copywriting - Frameworks

### AIDA (pages de vente)
```
Attention  → Titre accrocheur, stat choc
Intérêt    → Problème du lecteur
Désir      → Solution + preuves
Action     → CTA unique et clair
```

### PAS (articles blog)
```
Problème   → Identifier la douleur
Agitation  → Amplifier les conséquences
Solution   → Présenter la réponse
```

### Landing pages - checklist
- [ ] 1 seul objectif par page
- [ ] Headline = bénéfice principal
- [ ] Social proof (logos, témoignages, chiffres)
- [ ] CTA visible above the fold + répété
- [ ] Formulaire court (3-5 champs max)
- [ ] Page mobile-first

## Blog - Bonnes pratiques

### Structure article SEO
```
H1: Titre (keyword principal)
  Intro (hook + promesse, 150 mots max)
  H2: Sous-thème 1
    H3: Détails
  H2: Sous-thème 2
  H2: FAQ (questions longue traîne)
  Conclusion + CTA
```

### Checklist publication
- [ ] Titre < 60 caractères (SEO)
- [ ] Meta description < 155 caractères
- [ ] Images optimisées + alt text
- [ ] Liens internes (3-5 min)
- [ ] CTA en fin d'article
- [ ] URL slug propre et court
