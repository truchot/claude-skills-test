# Communication Templates

## Email de bienvenue

```markdown
Objet : Bienvenue [Prenom] - Votre projet [nom] demarre !

Bonjour [Prenom],

Nous sommes ravis de vous accompagner sur [projet].

Voici ce qui vous attend dans les prochains jours :
1. [Etape 1 - ex: Un appel de cadrage pour affiner vos besoins]
2. [Etape 2 - ex: La proposition detaillee avec planning]
3. [Etape 3 - ex: Le lancement officiel du projet]

Votre interlocuteur principal : [nom, role, email]

N'hesitez pas a nous solliciter a tout moment.

Bien cordialement,
[Signature]
```

## Rapport d'avancement hebdomadaire

```markdown
## Point projet - Semaine [N]

### Ce qui a ete realise cette semaine
- [Realisation 1 en langage client]
- [Realisation 2 en langage client]

### Ce qui est prevu la semaine prochaine
- [Action prevue 1]
- [Action prevue 2]

### Points qui necessitent votre attention
- [Element necessitant validation/decision]

### Indicateurs
- Avancement global : [X]%
- Prochaine etape cle : [jalon] prevu le [date]
- Budget consomme : [X]% du total

Bonne semaine,
[Signature]
```

## Communication de retard

```markdown
Objet : Point d'avancement [projet] - Ajustement planning

Bonjour [Prenom],

Je souhaitais vous informer d'un ajustement sur le planning de [projet].

**Ce qui s'est passe** : [explication simple, sans jargon]
**Impact** : [consequence concrete pour le client]
**Notre plan** : [actions correctives, nouveau delai]

Ce decalage de [X jours] n'impacte pas [element rassurant].

Je reste disponible pour en discuter de vive voix si vous le souhaitez.

Bien cordialement,
[Signature]
```

```markdown
## Bilan projet - [Nom]

### Resultats atteints
- [Objectif 1] : [resultat mesurable]
- [Objectif 2] : [resultat mesurable]

### Chiffres cles
| Indicateur | Avant | Apres | Evolution |
|------------|-------|-------|-----------|
| [KPI 1] | | | |
| [KPI 2] | | | |

### Prochaines etapes recommandees
1. [Recommandation 1]
2. [Recommandation 2]

### Votre avis compte
Nous serions ravis de recueillir votre retour :
- Satisfaction globale (1-10) : ___
- Ce qui vous a le plus plu : ___

```

## Regles de tonalite

| Contexte | Ton | Exemple |
|----------|-----|---------|
| Bonne nouvelle | Enthousiaste | "Excellente nouvelle !" |
| Avancement normal | Serein, professionnel | "Le projet avance bien." |
| Retard mineur | Transparent, rassurant | "Un leger ajustement..." |
| Probleme majeur | Empathique, solutioniste | "Nous comprenons l'impact..." |
| Demande de validation | Clair, facilitant | "Il nous faut votre retour sur..." |
