# Contexte Content Marketing

Connaissances essentielles pour la stratégie et production de contenu.

## Fondamentaux

### Le Content Marketing en Bref

```
Définition:
  Créer et distribuer du contenu pertinent et utile
  pour attirer et fidéliser une audience définie,
  dans le but de générer des actions profitables.

Objectifs:
  1. ATTIRER     → Trafic qualifié (SEO, social)
  2. ENGAGER     → Temps passé, interactions
  3. CONVERTIR   → Leads, ventes
  4. FIDÉLISER   → Récurrence, ambassadeurs
```

### Types de Contenu

```
Par format:
  Texte       → Articles, guides, ebooks, newsletters
  Visuel      → Infographies, photos, illustrations
  Vidéo       → Tutoriels, interviews, webinaires
  Audio       → Podcasts, audiobooks
  Interactif  → Quiz, calculateurs, outils

Par objectif funnel:
  TOFU (Awareness)   → Blog posts, infographies, vidéos sociales
  MOFU (Consideration) → Guides, comparatifs, webinaires, case studies
  BOFU (Decision)    → Démos, témoignages, essais gratuits
```

## Stratégie de Contenu

### Framework de Stratégie

```
1. AUDIT
   - Contenu existant (inventory)
   - Performance actuelle
   - Gaps vs concurrence

2. OBJECTIFS
   - Métriques business
   - KPIs content
   - Timeline

3. AUDIENCE
   - Personas
   - Pain points
   - Questions fréquentes
   - Parcours d'achat

4. THÉMATIQUES
   - Piliers de contenu (3-5)
   - Clusters thématiques
   - Mots-clés cibles

5. FORMATS & CANAUX
   - Types de contenu prioritaires
   - Canaux de distribution
   - Fréquence de publication

6. PRODUCTION
   - Process éditorial
   - Ressources (interne/externe)
   - Calendrier

7. MESURE
   - KPIs par type de contenu
   - Reporting cadence
   - Optimisation continue
```

### Piliers de Contenu (Content Pillars)

```
Exemple SaaS B2B:

PILIER 1: Éducation produit
  → Tutoriels, guides d'utilisation, FAQ
  → Objectif: Onboarding, support

PILIER 2: Thought leadership
  → Tendances, opinions, études
  → Objectif: Autorité, backlinks

PILIER 3: Success stories
  → Case studies, témoignages
  → Objectif: Preuve sociale, conversion

PILIER 4: Actualités secteur
  → News, analyses, événements
  → Objectif: Engagement, récurrence
```

## Production de Contenu

### Brief Éditorial

```markdown
# Brief: [Titre de l'article]

## Objectif
- But de l'article
- Action attendue du lecteur

## Cible
- Persona concerné
- Niveau de connaissance (débutant/intermédiaire/expert)

## SEO
- Mot-clé principal:
- Mots-clés secondaires:
- Volume de recherche:
- Intention de recherche: (informationnelle/transactionnelle/navigationnelle)

## Structure proposée
- H1:
- H2: [sections principales]
- Longueur cible: X mots

## Sources / Références
- [liens études, données, inspirations]

## CTA
- Action souhaitée en fin d'article

## Deadline
- Premier draft:
- Publication:
```

### Structure Article Optimisée

```
1. TITRE (H1)
   - Inclut le mot-clé principal
   - Promet une valeur claire
   - 50-60 caractères

2. INTRODUCTION (150-200 mots)
   - Accroche (problème/question)
   - Contexte rapide
   - Promesse de l'article
   - Preview de la structure

3. CORPS (sections H2/H3)
   - 1 idée principale par section
   - Paragraphes courts (3-4 phrases)
   - Sous-titres descriptifs
   - Listes à puces pour la lisibilité
   - Exemples concrets
   - Visuels (images, schémas, tableaux)

4. CONCLUSION
   - Résumé des points clés
   - Réponse à la promesse initiale
   - CTA clair

5. ÉLÉMENTS BONUS
   - FAQ (pour Featured Snippets)
   - Table des matières (articles longs)
   - Auteur + date mise à jour
```

### Checklist Rédaction

```
Avant rédaction:
  □ Brief validé
  □ Recherche mots-clés faite
  □ Intention de recherche identifiée
  □ Top 5 SERP analysé
  □ Sources collectées

Pendant rédaction:
  □ Mot-clé dans H1, intro, H2
  □ Champs sémantiques couverts
  □ Liens internes pertinents
  □ Données sourcées
  □ Exemples concrets

Avant publication:
  □ Relecture orthographe
  □ Meta title et description
  □ URL optimisée
  □ Images compressées + alt
  □ CTA présent
  □ Catégorie/tags assignés
  □ Preview mobile OK
```

### Fréquences Recommandées

```
Par canal:

Blog:
  Minimum viable → 1 article/semaine
  Idéal         → 2-4 articles/semaine
  (Qualité > quantité)

Newsletter:
  B2B           → 1x/semaine ou 2x/mois
  B2C           → 1-2x/semaine
  E-commerce    → 2-3x/semaine

Réseaux sociaux:
  LinkedIn (B2B)    → 3-5x/semaine
  Instagram         → 1x/jour
  Twitter/X         → 3-5x/jour
  TikTok            → 1-3x/jour
```

### Workflow Éditorial

```
PLANIFICATION (J-14)
  → Idées brainstorm
  → Validation thèmes
  → Assignation rédacteurs
  → Briefs créés

PRODUCTION (J-7 à J-3)
  → Recherche / interviews
  → Rédaction draft
  → Révision interne

VALIDATION (J-3 à J-1)
  → Relecture éditoriale
  → Validation SEO
  → Validation légale (si nécessaire)
  → Préparation publication

PUBLICATION (J)
  → Mise en ligne
  → Partage réseaux sociaux
  → Newsletter (si applicable)

POST-PUBLICATION (J+7)
  → Suivi performance
  → Indexation vérifiée
  → Engagement réseaux
```

## Formats de Contenu

### Article de Blog

```
Types d'articles:
  How-to guide     → "Comment faire X en 5 étapes"
  Listicle         → "10 outils pour X"
  Comparatif       → "X vs Y : lequel choisir ?"
  Ultimate guide   → "Le guide complet du X"
  Newsjacking      → "X annonce Y : ce que ça change"
  Opinion/POV      → "Pourquoi X va disparaître"
  Case study       → "Comment X a augmenté Y de Z%"
  Interview        → "Rencontre avec X, expert en Y"

Longueur par type:
  News/actualité   → 500-800 mots
  How-to simple    → 1000-1500 mots
  Guide approfondi → 2000-3000 mots
  Ultimate guide   → 3000-5000+ mots
```

### Lead Magnet (Gated Content)

```
Formats populaires:
  - Ebook / Guide PDF
  - Template / Checklist
  - Calculateur / Outil
  - Webinaire replay
  - Mini-cours email
  - Étude / Rapport

Landing page structure:
  1. Headline bénéfice
  2. Bullet points valeur
  3. Preview visuelle
  4. Formulaire court
  5. Preuve sociale

Règle: Valeur perçue > effort demandé
```

### Case Study

```
Structure STAR:
  S - Situation   → Contexte client, problème
  T - Task        → Objectifs fixés
  A - Action      → Solution mise en place
  R - Results     → Résultats chiffrés

Template:
  1. Résumé (30 sec read)
     - Client: [nom, secteur, taille]
     - Défi: [problème principal]
     - Résultat: [chiffre clé]

  2. Contexte (2-3 paragraphes)
     - Qui est le client
     - Situation avant

  3. Défi (2-3 paragraphes)
     - Problème précis
     - Enjeux business
     - Tentatives précédentes

  4. Solution (3-5 paragraphes)
     - Approche choisie
     - Mise en œuvre
     - Timeline

  5. Résultats (bullet points)
     - Métriques avant/après
     - ROI si possible
     - Quote client

  6. CTA
```

### Newsletter

```
Structure efficace:
  1. Objet email (40-50 caractères)
     - Curiosité ou bénéfice clair
     - Éviter spam words

  2. Preview text (80-100 caractères)
     - Complète l'objet
     - Incite à ouvrir

  3. Corps
     - 1 sujet principal
     - Paragraphes courts
     - 1 CTA clair

  4. Footer
     - Lien désinscription
     - Coordonnées
```

## Mesure de Performance

### KPIs Content

```
AWARENESS (TOFU)
  - Pageviews
  - Unique visitors
  - Impressions
  - Reach social

ENGAGEMENT (MOFU)
  - Time on page
  - Scroll depth
  - Comments
  - Shares
  - Saves

CONVERSION (BOFU)
  - Lead gen (downloads, signups)
  - Conversion rate
  - Attribution revenue

QUALITY
  - Bounce rate
  - Pages/session
  - Return visitors
```

### Content Scoring

```
Score par article (exemple):

Trafic:
  < 100 vues/mois    → 1 pt
  100-500 vues/mois  → 2 pts
  500-1000 vues/mois → 3 pts
  > 1000 vues/mois   → 4 pts

Engagement:
  Bounce > 80%       → 0 pt
  Bounce 60-80%      → 1 pt
  Bounce < 60%       → 2 pts

Conversion:
  0 conversion       → 0 pt
  1-5 conversions    → 1 pt
  > 5 conversions    → 2 pts

Score total: /8
  7-8 → Top performer (promouvoir)
  4-6 → Performer (optimiser)
  1-3 → Underperformer (réviser ou supprimer)
```
