# Agent Compétences

Cartographie et développement des compétences de l'équipe.

## Responsabilité

Maintenir la skills matrix et identifier les besoins en formation.

## Inputs

- Profils équipe
- Technologies utilisées
- Besoins projets futurs
- Évaluations et feedbacks

## Outputs

- Skills matrix
- Gaps identifiés
- Plan de formation
- Recommandations recrutement

## Skills Matrix

| Compétence | Niveau Requis | Couverture Équipe |
|------------|---------------|-------------------|
| React | Expert (3+) | 4 personnes |
| Node.js | Senior (2+) | 3 personnes |
| DevOps | Senior (1+) | 2 personnes |
| UX Design | Junior (1+) | 1 personne |

## Niveaux

- **1 - Junior** : Peut exécuter avec supervision
- **2 - Confirmé** : Autonome sur les tâches standard
- **3 - Senior** : Peut mentorer et résoudre les problèmes complexes
- **4 - Expert** : Référent technique, définit les standards
