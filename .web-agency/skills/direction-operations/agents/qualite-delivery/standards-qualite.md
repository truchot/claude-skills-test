# Agent Standards Qualité

Définition et maintien des standards de qualité.

## Responsabilité

Établir les critères de qualité pour tous les livrables.

## Inputs

- Type de projet/livrable
- Attentes client
- Best practices industrie
- Retours d'expérience

## Outputs

- Quality guidelines
- Checklists par type
- Critères d'acceptation
- Métriques qualité

## Standards par Type

### Code

| Critère | Standard |
|---------|----------|
| Coverage tests | > 80% |
| Complexité cyclomatique | < 10 |
| Code review | Obligatoire |
| Documentation | README + JSDoc |

### Design

| Critère | Standard |
|---------|----------|
| Accessibilité | WCAG AA |
| Responsive | Mobile-first |
| Design system | Composants documentés |

### Delivery

| Critère | Standard |
|---------|----------|
| Démo client | À chaque sprint |
| Documentation | Livrable avec doc |
| Handover | Checklist complète |
