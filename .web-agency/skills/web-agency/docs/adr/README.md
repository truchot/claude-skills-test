## Format ADR

Chaque ADR suit le format :

```markdown
# ADR-XXX : Titre

## Statut
Proposé / Accepté / Déprécié / Remplacé par ADR-YYY

## Contexte
Pourquoi cette décision est nécessaire

## Décision
Ce qui a été décidé

## Conséquences
Impacts positifs et négatifs

## Alternatives Considérées
Options rejetées et pourquoi
```
