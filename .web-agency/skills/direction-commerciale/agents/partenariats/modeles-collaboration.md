# Agent Modèles Collaboration

Frameworks et modèles de collaboration avec les partenaires.

## Responsabilité

Définir les modes de collaboration avec les partenaires.

## Inputs

- Type de partenaire
- Objectifs communs
- Ressources disponibles
- Contraintes

## Outputs

- Framework de collaboration
- Template d'accord
- Process de travail
- KPIs de suivi

## Modèles de Collaboration

### 1. Référencement Mutuel

```
Nous → Leads qualifiés → Partenaire
Partenaire → Leads qualifiés → Nous

Commission : 10-15% du CA généré
```

### 2. Sous-traitance

```
Nous (prime) → Projet → Partenaire (sous-traitant)

Marge : 15-25% sur jours revendus
```

### 3. Co-réalisation

```
Nous + Partenaire → Projet commun

Répartition : Au prorata des jours
```

### 4. White Label

```
Partenaire → Vend sous sa marque → Notre prestation

Marge partenaire : 20-30%
```

## Template Accord Partenariat

```
1. Objet et périmètre
2. Obligations de chaque partie
3. Conditions commerciales
4. Propriété intellectuelle
5. Confidentialité
6. Durée et résiliation
7. Règlement des litiges
```
