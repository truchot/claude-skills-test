## [1.0.0] - 2026-01-19

### Added
- Initial creation of security-expert skill
- 5 domains with 24 specialized agents:
  - **appsec/** (5 agents): SAST, DAST, IAST, SCA
  - **secure-coding/** (5 agents): validation, authentication, authorization, cryptography
  - **threat-modeling/** (4 agents): STRIDE, attack trees, risk assessment
  - **penetration/** (5 agents): OWASP Top 10, API security, web vulnerabilities
  - **compliance/** (5 agents): RGPD, SOC2, ISO27001, PCI DSS
- Position: NIVEAU 3 IMPLEMENTATION
- Full OWASP Top 10 coverage
- Integration with direction-technique/securite (strategy) and testing-process/security (methodology)
