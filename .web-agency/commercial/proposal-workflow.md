**IMPORTANT** : Toujours indiquer une date limite de validité.

| 10 000 - 50 000 € | 45 jours |
| > 50 000 € | 60 jours |

> Passé ce délai, le devis n'engage plus l'agence (protection juridique).

## Étape 2 : Présentation

### 2.1 Règle d'Or

> **TOUJOURS présenter le devis de vive voix** (visio ou présentiel).
> Ne jamais envoyer un devis "à froid" par email.

### 2.2 Déroulé Type (45-60 min)

| Phase | Durée | Contenu |
|-------|-------|---------|
| **Rappel contexte** | 5 min | Reformuler le besoin, montrer qu'on a compris |
| **Solution proposée** | 15 min | Parcourir le cahier des charges, expliquer l'approche |
| **Chiffrage détaillé** | 15 min | Justifier chaque lot, expliquer la valeur |
| **Planning** | 5 min | Présenter les jalons, dates clés |
| **Conditions** | 5 min | Paiement, validité, prochaines étapes |
| **Questions** | 10 min | Répondre aux interrogations |

### 2.3 Arguments par Lot

Préparer pour chaque lot :
- **Ce qu'il contient** (en langage simple)
- **Pourquoi c'est nécessaire** (valeur business)
- **Ce qui justifie le prix** (complexité, expertise)

### 2.4 Objections Fréquentes

| Objection | Réponse type |
|-----------|--------------|
| "C'est trop cher" | Recentrer sur la valeur, pas le coût. Proposer de prioriser. |
| "Concurrent moins cher" | Demander le détail, comparer à périmètre égal. |
| "Peut-on réduire ?" | Proposer de retirer du scope (Options → Won't). |
| "Délai trop long" | Expliquer les dépendances, risques d'aller plus vite. |
| "On réfléchit" | Fixer un RDV de suivi, ne pas laisser dans le vague. |

## Étape 3 : Négociation

### 3.1 Marges de Manœuvre

Définir EN AMONT les limites de négociation :

| Levier | Marge autorisée | Validation requise |
|--------|-----------------|-------------------|
| Remise globale | Jusqu'à -10% | Commercial |
| Remise globale | -10 à -20% | Direction |
| Retrait de scope | Libre | Commercial |
| Étalement paiement | 4 échéances max | Commercial |
| Report démarrage | 3 mois max | Commercial |

### 3.2 Stratégies de Négociation

**Si demande de remise** :
1. D'abord proposer de retirer du scope (gagnant-gagnant)
2. Si impossible, proposer une remise contre contrepartie :
   - Paiement comptant → -5%
   - Engagement maintenance → -5%
   - Témoignage client → -3%
   - Référencement portfolio → -2%

**Si budget insuffisant** :
1. Proposer un MVP (Minimum Viable Product)
2. Phaser le projet (V1 → V2)
3. Proposer des options à activer plus tard

### 3.3 Avenant

Si modifications après présentation :

## Étape 4 : Issue - Acceptation

### 4.1 Signature du Devis

Le client doit :
1. Parapher chaque page
2. Écrire "Bon pour accord" sur la dernière page
3. Dater et signer
4. Retourner le devis signé (scan acceptable, original à suivre)

### 4.2 Confirmation de Commande

À réception du devis signé, envoyer :

```
Objet : Confirmation de commande - [Projet]

Bonjour [Prénom],

Nous accusons réception de votre bon pour accord sur le devis n°[XXX].

Voici les prochaines étapes :
1. Facture d'acompte en pièce jointe (30% = XXX € TTC)
2. À réception du paiement, nous programmons le kick-off
3. RDV de lancement proposé : [Date]

Merci pour votre confiance.

Cordialement,
[Signature]

PJ : Facture d'acompte FA-[XXX]
```

### 4.3 Checklist Avant Démarrage

- [ ] Devis signé reçu
- [ ] Facture d'acompte envoyée
- [ ] Acompte encaissé
- [ ] Kick-off planifié
- [ ] Accès/contenus demandés au client
- [ ] Équipe projet informée
- [ ] Dossier projet créé (`.project/`)

## Étape 4 bis : Issue - Refus

### 4.1 Réponse au Refus

**Objectif** : Garder la porte ouverte pour le futur.

```
Objet : Merci pour votre retour - [Projet]

Bonjour [Prénom],

Merci de nous avoir informés de votre décision.

Nous sommes ravis que vous ayez trouvé une solution qui correspond
à vos attentes. Si le contexte évolue ou si vous avez besoin d'un
regard extérieur sur votre projet, n'hésitez pas à nous solliciter.

Nous serions également intéressés de voir le résultat une fois
le projet abouti - nous suivons toujours avec curiosité les projets
sur lesquels nous avons travaillé, même de loin.

Bonne continuation dans vos projets.

Cordialement,
[Signature]
```

### 4.2 Feedback (optionnel)

Si relation le permet, demander :
- Raison principale du refus
- Ce qui aurait pu faire la différence
- Garder le contact LinkedIn

### 4.3 Suivi CRM

- [ ] Marquer opportunité comme "Perdue"
- [ ] Noter la raison du refus
- [ ] Programmer un rappel dans 6-12 mois
- [ ] Ajouter à la newsletter (si accord)

## Étape 5 : Acompte

### 5.1 Facture d'Acompte

| Élément | Valeur |
|---------|--------|
| Référence | FA-[Année]-[N°] |
| Objet | Acompte 30% - Projet [Nom] |
| Base | Référence devis n°[XXX] du [Date] |
| Montant HT | XXX € |
| TVA 20% | XXX € |
| **Total TTC** | **XXX €** |
| Échéance | À réception |

### 5.2 Conditions de Démarrage

**RÈGLE ABSOLUE** : Aucun travail ne commence avant encaissement de l'acompte.

| Acompte reçu | Action |
|--------------|--------|
| ✅ Oui | Lancer le projet |
| ❌ Non | Relancer le client |

### 5.3 Relance Acompte

Si non-paiement après 7 jours :

```
Objet : Relance facture d'acompte FA-[XXX]

Bonjour [Prénom],

Sauf erreur de notre part, nous n'avons pas encore reçu le règlement
de la facture d'acompte n°FA-[XXX] d'un montant de XXX € TTC.

Pour rappel, le démarrage du projet est conditionné à la réception
de cet acompte.

Afin de respecter le planning prévu (lancement semaine [X]),
pourriez-vous effectuer le virement dans les meilleurs délais ?

Coordonnées bancaires en pièce jointe.

Cordialement,
[Signature]
```
