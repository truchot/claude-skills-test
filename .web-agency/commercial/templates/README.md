# Templates Emails Commerciaux

> Emails types pour le cycle avant-projet.

## Liste des Templates

| Template | Usage | Quand |
|----------|-------|-------|
| `confirmation-rdv.md` | Confirmer un RDV découverte | Après prise de RDV |
| `envoi-cdc.md` | Envoyer le cahier des charges | Après rédaction CDC |
| `envoi-devis.md` | Envoyer le devis | Après présentation orale |
| `relance-devis.md` | Relancer sans réponse | J+7 après envoi |
| `confirmation-commande.md` | Confirmer la commande | À réception devis signé |
| `relance-acompte.md` | Relancer le paiement | J+7 après facture |
| `refus-suivi.md` | Répondre à un refus | À réception du refus |

## Règles d'Utilisation

1. **Personnaliser** : Adapter le ton au client
2. **Relire** : Vérifier noms, montants, dates
3. **Pièces jointes** : Vérifier qu'elles sont jointes
4. **Objet clair** : Permet de retrouver l'email facilement

## Variables à Remplacer

| Variable | Description |
|----------|-------------|
| `[Prénom]` | Prénom du contact |
| `[Projet]` | Nom du projet |
| `[Date]` | Date concernée |
| `[N°]` | Numéro de document |
| `[XXX €]` | Montant |
